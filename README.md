# 이현령

> 저는 이현령입니다.

**이현령**은 AI 기반 서비스, RAG 시스템, 문제 해결에 관심을 두고 활동하는 개발자이다. 중학교 시절 C 언어를 접하며 프로그래밍을 시작했고, 대전대신고등학교 개발자 동아리 A.C.T와 건국대학교 컴퓨터공학과를 거쳐 Microsoft AI School에서 AI 서비스 개발 프로젝트를 수행했다. 2025년부터 GraphAI에서 AI를 위한 복합 DB와 RAG 시스템을 개발하고 있다.

## 기본 정보

이현령은 프로그래밍 언어와 AI 기반 서비스 개발에 관심을 가지고 있으며, 커뮤니티 운영과 문제 해결 활동을 통해 학습과 협업 경험을 쌓았다. Python을 주력 언어로 사용하며, LLM 기반 애플리케이션과 영상 처리 시스템을 중심으로 프로젝트를 수행했다.

## 관심 분야

- **인공지능 기반 서비스 개발**: LLM, RAG, 에이전트 구조를 활용한 서비스 구현
- **컴퓨터 비전**: 객체 탐지, 이미지 분류, 행동 분석 모델 개발
- **프로그래밍 언어**: Python을 중심으로 한 언어 학습과 활용
- **문제 해결**: 온라인 저지와 스터디를 통한 알고리즘 및 자료구조 학습
- **커뮤니티**: 학습 공동체와 온라인 커뮤니티 운영
